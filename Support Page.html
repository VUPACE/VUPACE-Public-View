VUPACE Overview

Back to Top

Tabs on the Navigation Bar

  • Logo  – Clicking this will take you to the home page of VUPACE. The Home Page is the page you first encounter when accessing the webpage.
  • Programs – This is a list of all active or recently active programs in VUPACE. Here you can find information related to courses and specific programs.
  • Certificates - This is where you can search and enroll in a bundle of courses for a certificate. 
  • Courses – This is where you can search and enroll in an available course. See the Enrolling in a Course VUPACE User Guide below.
  • Contact – This is a list of all program contacts. You can utilize this page if you have questions about a course or application.
  • Support – This is where you can access guides, like this one, to help navigate the system.

In the top middle of the page above, there is a Shopping Cart icon.

  • This is your Shopping Cart and contains any courses or additional items you have added to your basket.

In the top middle of this page, next to the shopping cart, there is a login dropdown that provides the options: Student Login and Instructor Login. Select Student Login to login or create an account. Note: You can access the student portal at any time by clicking your name in the top navigation bar.
For more information on creating an account, please see Creating an Account VUPACE User Guide below.

In the top left of the page, there is a Course Search Bar where you can search for a course.

You can resend the password to your email by clicking Forgot Password on the log in page and following the prompts. For further information, see the Resetting your Password VUPACE User Guide above.

First, check your spam folder and be sure to wait at least 5 minutes. If the email still hasn’t come through, it’s possible that the email you used to set up your account  was incorrect. Send an email to pace@vanderbilt.edu with your first name, last name, your preferred email, and a brief description of the problem with the subject line Account Help.  

If the application is required for enrollment, you must complete it. Please direct any further questions to your Program Contact

Please contact your Program Contact

VUPACE operates best on a browser like Chrome on a computer or laptop. Please access the website using these configurations.

If you have just enrolled in a course, please wait 20-30 minutes before accessing the Brightspace link. This gives the system time to properly enroll you in the course. If you wait this amount of time and you still encounter the error message, send an email to pace@vanderbilt.edu with your first name, last name, your preferred email, and a brief description of the problem with the subject line Brightspace Account Help.

Please contact your Program Contact.

The course will have a Join Waitlist option. You will have the opportunity to enroll in the course if someone drops from the course. Please direct any further questions to your Program Contact.
